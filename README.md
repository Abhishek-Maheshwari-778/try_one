## Achievement List

| Achievement Badge | Description | How to Unlock (Steps) |
| :--- | :--- | :--- |
| **Quick Draw** | Closing an issue within 5 minutes of opening it. | 1. Create a new issue.<br>2. Wait 10 seconds.<br>3. Close it (under 5 mins). |
| **Pull Shark** | Merging Pull Requests (PR). | 1. Edit a file.<br>2. Create a new branch.<br>3. Create PR.<br>4. Merge PR. |
| **Pair Extraordinaire** | Co-authoring a commit. | Add `Co-authored-by: username <email>` to commit message. |
| **Dynamic Duo** | Merging a PR reviewed by someone else. | 1. Create PR.<br>2. Request review from a friend.<br>3. Merge after approval. |
| **Galaxy Brain** | 2 Accepted Answers in Discussions. | Answer questions in discussions and get marked as "Answer". |
| **YOLO** | Merging a PR without code review. | Create PR and merge immediately without review. |
| **Starstruck** | Creating a popular repository. | Get 16 stars. |
| **Public Sponsor** | Sponsoring an open-source project. | Sponsor a developer. |
| **Heart on Your Sleeve** | Using reactions (emojis) on GitHub. | React to a comment with an emoji. |

## Automation Status

- **Pair Extraordinaire**: This file was committed with a co-author to trigger this.
- **Pull Shark / YOLO**: Branches have been pushed. Go to GitHub and merge the PRs.
