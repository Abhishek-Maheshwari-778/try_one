Follow these exact steps to unlock the remaining badges.

### 1. Dynamic Duo (Requires a Friend)
**Goal**: Merge a PR that was approved by someone else.
1. Go to this repo's **Pull Requests** tab.
2. Click **New Pull Request**.
3. Select `feature/dynamic-duo` (or create a new branch like `patch-1`).
4. Create the PR.
5. **CRITICAL**: Copy the link and send it to a friend. Ask them to "Approve" the changes.
6. Once they approve, click **Merge**.

### 2. Galaxy Brain (Requires 2 Accepted Answers)
**Goal**: Get your answers marked as "Accepted" in a Discussion.
1. Go to **Settings** > **General** > **Features** and enable **Discussions**.
2. Go to the newly visible **Discussions** tab.
3. Create a "New Discussion" (Category: Q&A). Title: "How do I get Galaxy Brain?". body: "Help me."
4. **Switch to a second account** (or ask a friend).
5. Reply to the discussion: "You need 2 accepted answers."
6. Using the *original* account, click the **Checkmark (‚úì)** icon on the reply to mark it as the Answer.
7. Repeat this process 2 times.

### 3. Starstruck (Requires 16 Stars)

3. You need 16 unique people to click the **Star** button.
